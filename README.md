### 环境

这个工程使用的环境在`keil5`,`keil4`需要自己测试。

### 工程

`bin` ---------------------------------------------------------`hex文件存放目录`

`adcManager` ---------------------------------------------------------`adc转换，光敏，温湿度等转换函数`

`common` ---------------------------------------------------------`一些自定义的数据类型`

`globalManager` ---------------------------------------------------------`全局管理函数`

`gsmManager` ---------------------------------------------------------`gsm模块函数`

`lcdManager` ---------------------------------------------------------`1602模块函数`

`linkFile` ---------------------------------------------------------`编译中间连接文件`

`MotorManager` ---------------------------------------------------------`电机函数`

`main.c` ---------------------------------------------------------`main函数`

### 引脚

**这个引脚是单片机接线的方式，不要随意更改，因为所有的编程都是基于这些引脚编写。**
